# Daily Health & Habit Tracker (Excel)

## Overview
This project is an Excel-based Daily Health and Habit Tracker designed to
monitor personal habits and overall consistency across a month.
It helps visualize daily progress, weekly performance, and monthly completion
percentage using checkboxes, formulas, and charts.

## Features
- Daily habit tracking using cell-based checkboxes
- Automatic calculation of habit completion percentage
- Weekly and monthly progress analysis
- Visual dashboards with charts for quick insights

## Habits Tracked
Some of the habits included:
- Wake up at 5 AM
- Workout & stretching
- Meditation
- Skincare & haircare
- Coding practice
- Academic study
- Healthy eating habits (protein intake, no junk food)
- Sleep routine

## How It Works
- Each habit is tracked daily using checkboxes
- Checked boxes are treated as completed tasks
- Excel formulas are used to:
  - Count completed tasks per day
  - Calculate habit-wise completion percentage
  - Compute overall monthly progress
- Data is summarized week-wise and month-wise automatically

## Data Analysis & Logic
The tracker uses Excel formulas such as:
- Logical functions
- Counting functions 
- Percentage calculations
- Cell referencing across weeks

Exact formulas may vary across sheets, but the logic focuses on
task completion and consistency tracking.

## Visualizations
The project includes:
- Pie chart showing completed vs remaining percentage
- Line chart for daily progress trend
- Bar chart for habit-wise completion comparison

These charts update automatically based on checkbox input.

## Use Case
- Personal productivity tracking
- Health & fitness habit monitoring
- Consistency analysis over time
- Beginner-friendly data analysis project using Excel

This project demonstrates practical use of Excel for automation,
data analysis, and personal productivity tracking.
